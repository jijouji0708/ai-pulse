# AI Pulse フィロソフィー

## 社是
**「正直な機械と、考える人間で。」**

## フィロソフィー
テクノロジーの恩恵は、それを理解できる人だけのものであってはならない。
私たちはAIの力で情報を届け、人間の判断で品質を守る。隠さない、飾らない、止まらない——この3つの約束が、AI Pulseのすべての行動の土台である。

## ミッション
AI・テクノロジーの最前線を、正直なAIと責任ある人間の協働で、毎日・日本語で届ける。「AIが作った」を隠すのではなく、「AIだからこそできる速度と網羅性」を証明する。

## ビジョン
AI運営メディアの世界標準となり、「透明であることが最強の競争力になる」ことを実証する。

## バリュー
1. **Glass Box（ガラスの箱）** — プロセスも失敗もすべて公開する。ブラックボックスの逆を行く。
2. **毎日、少し良く** — 自己改善ループを回し続ける。昨日の自分に勝つことだけに集中する。
3. **人間が舵、AIがエンジン** — 戦略判断は人間が握る。AIは最高の実行力を提供する。それぞれの強みを混ぜない。
4. **読者の時間を奪わない** — 量で埋めない。1記事で「読んでよかった」を必ず1つ残す。
5. **コスト正直** — 運営コストを公開し、再現可能なモデルを示す。「すごい」ではなく「自分もできそう」を届ける。

## 検討プロセス

### サイクル1: 初案

#### 社是（初案）
**「AIに正直を。情報に自由を。」**

選定理由：AI Pulseの二大柱である「透明性」と「情報アクセスの民主化」を一言に凝縮した。AI生成を隠さないという姿勢（正直）と、テクノロジー情報を誰にでも届けるという使命（自由）を対で表現。

#### フィロソフィー（初案）
「テクノロジーの恩恵は、一部の専門家だけのものであってはならない。私たちはAIの力で情報を届け、人間の知恵で品質を守る。完全な自律ではなく、正直な協働こそが、持続可能なAIメディアの姿だと信じる。」

選定理由：CMU研究の「完全自律はまだ困難」という教訓と、「人間が戦略判断、AIが実行」のハイブリッドモデルを哲学レベルで表現。The Rundown AIの「AIの民主化」と方向性は似るが、「協働」に軸足を置くことで差別化。

#### ミッション（初案）
「AI・テクノロジーの最前線を、毎日、日本語で、透明なAIと人間の協働で届ける。」

選定理由：What（AI/テクノロジー情報）、How（AI+人間の協働、透明性）、When（毎日）、Who（日本語読者）を一文に収めた。

#### ビジョン（初案）
「AI運営メディアの世界的な成功モデルとなり、透明性が信頼と競争力を生むことを証明する。」

選定理由：「成功モデル」＝再現可能性を示す。「透明性→信頼→競争力」の因果関係を明示。

#### バリュー（初案）
1. **透明性ファースト** — AI生成を全記事に明記し、運営の裏側も公開する
2. **毎日進化** — 自己改善ループ（Karpathyパターン）で品質を継続向上する
3. **人間×AI協働** — 人間が戦略、AIが実行。混ぜない、任せきらない
4. **読者価値最優先** — 量より質。1記事で必ず1つの「知ってよかった」を届ける
5. **コスト透明性** — 運営コストを公開し、再現可能な運営モデルを示す

### サイクル2: 批判的レビュー

#### 社是の問題点
- 「AIに正直を。情報に自由を。」→ **「情報に自由を」がThe Rundown AIの「AIの民主化」と被る。** また、やや抽象的で、AI Pulse固有の強みが見えにくい。
- 「正直」は良いキーワードだが、「自由」は他社も言える。AI Pulseだけが言えることは何か？→ **「機械（AI）と人間の協働」** がユニーク。
- 修正案：**「正直な機械と、考える人間で。」** → AI（機械）の透明性と人間の判断力の両方を一言で表現。「機械」とあえて言い切ることで、AIを神格化せず道具として正直に使う姿勢を示す。

#### フィロソフィーの問題点
- 「一部の専門家だけのものであってはならない」→ やや説教調。The Rundown AIの「民主化」メッセージとも近い。
- 「完全な自律ではなく」→ 否定形から入るのは弱い。
- 修正案：「理解できる人だけのもの」に短縮し、「隠さない、飾らない、止まらない」という3つの行動原則を加えて具体性を持たせる。

#### ミッションの問題点
- 概ね良いが、「透明なAI」がやや意味不明。AIが透明なのではなく、AI利用を透明にしている。
- 「AIが作った」を隠すのではなく強みとして打ち出すニュアンスを追加すべき。Sports Illustratedの失敗と明確に対比させる。
- 修正案：「正直なAI」に変更し、「AIだからこそできる速度と網羅性を証明する」を追加。

#### ビジョンの問題点
- 「世界的な成功モデル」→ 現時点でゼロからのスタートなのに「世界的」は綺麗事すぎる。
- 「世界標準」に変更。メディア運営の標準を作るという意味。規模ではなく基準を示す存在になる、と読める。
- 修正案：「透明であることが最強の競争力になる」に変更。「最強」は強い言葉だが、信念として掲げる分には乖離しない。

#### バリューの問題点
1. **透明性ファースト** → 名前が平凡。どの企業でも言える。→ **「Glass Box（ガラスの箱）」** に変更。ブラックボックスの対義語として記憶に残る。
2. **毎日進化** → 「進化」が大げさ。→ **「毎日、少し良く」** に変更。Mindstreamの650日連続配信の地道さを反映。
3. **人間×AI協働** → 「協働」は抽象的。→ **「人間が舵、AIがエンジン」** に変更。役割分担が明確にイメージできる。
4. **読者価値最優先** → 概ね良いが、「AI Slop」問題への回答を明示すべき。→ **「読者の時間を奪わない」** に変更。量産低品質の逆を明確に。
5. **コスト透明性** → バリュー1の「Glass Box」と重複感。しかしコスト公開はAI Pulseの核心的差別化なので残す。→ **「コスト正直」** に名称変更し、「自分もできそう」という再現可能性のメッセージを強化。

#### エージェントの判断指針としての検証
各バリューが「エージェントが迷ったときの判断基準」になるか？
- Glass Box: 「この情報を隠すべきか？」→ 公開する（明確）
- 毎日、少し良く: 「大きな改修か小さな改善か？」→ 小さな改善を毎日（明確）
- 人間が舵、AIがエンジン: 「この判断をAIだけで決めていいか？」→ 戦略判断は人間にエスカレーション（明確）
- 読者の時間を奪わない: 「この記事を出すべきか？」→ 価値がなければ出さない（明確）
- コスト正直: 「このツールを導入すべきか？」→ コストを明示して判断（明確）

→ すべて行動指針として機能する。合格。

### サイクル3: 最終決定の理由

#### 社是：「正直な機械と、考える人間で。」
- 「正直な機械」でAIの透明性を、「考える人間」でハイブリッドモデルを表現
- 「機械」とあえて呼ぶことで、AIを過度に擬人化・神格化しない姿勢を示す
- Sports Illustratedの失敗（隠蔽）の逆を一言で宣言している
- 句点「。」で終わることで、宣言・約束のトーンを持たせた
- 日本語として短く、リズムがあり、記憶に残りやすい（7+7のリズム）

#### フィロソフィー
- 3文構成：理想（1文目）→ 手段（2文目）→ 行動原則（3文目）
- 「隠さない、飾らない、止まらない」の三拍子で記憶に残るフレーズを埋め込んだ
- 「隠さない」＝透明性、「飾らない」＝AI Slop排除、「止まらない」＝毎日配信の継続性

#### ミッション
- 「正直なAI」で透明性を、「責任ある人間」でハイブリッドモデルを再確認
- 「AIだからこそできる速度と網羅性」で、AI利用を弱みではなく強みとして定義
- 「証明する」という動詞で、実証プロジェクトとしての性質を明示

#### ビジョン
- 「世界標準」は規模ではなく基準。小さくても標準を作れる
- 「透明であることが最強の競争力になる」は、AI好感度26%の現実に対する回答
- 消費者がAI生成を嫌う理由は「品質」ではなく「不信」。透明性で不信を除去する戦略

#### バリュー
1. **Glass Box** — 「透明性」より具体的で視覚的。ブラックボックスAIへのアンチテーゼ
2. **毎日、少し良く** — カイゼン文化とKarpathyパターンの融合。過大な約束をしない誠実さ
3. **人間が舵、AIがエンジン** — CMU研究の教訓（完全自律の限界）を行動指針に昇華
4. **読者の時間を奪わない** — AI Slop問題への明確な回答。量ではなく質で勝負する宣言
5. **コスト正直** — Pieter Levelsのミニマル哲学を継承。再現可能性がメディアとしての信頼につながる

### 却下した代替案

#### 社是の代替案
| 代替案 | 却下理由 |
|---|---|
| 「AIに正直を。情報に自由を。」 | 「情報の自由」がThe Rundown AIの「民主化」と被る。AI Pulse固有の差別化が弱い |
| 「透明なAIで、確かな情報を。」 | 「確かな情報」が新聞社的で、テクノロジーメディアの先進性が感じられない |
| 「AIが書く。人間が選ぶ。」 | 分かりやすいが、「選ぶ」だけでは人間の役割が矮小化される。戦略判断の重みが伝わらない |
| 「嘘をつかないAIメディア。」 | 直球すぎて品がない。また「嘘をつかない」は最低ラインであり、理念として低い |

#### フィロソフィーの代替案
| 代替案 | 却下理由 |
|---|---|
| 「AIは道具であり、パートナーではない」 | 7体のエージェントに「チーム」としてのアイデンティティを与えたい方針と矛盾する |
| 「速さと正確さを両立する」 | 凡庸。どのメディアでも言える |

#### ビジョンの代替案
| 代替案 | 却下理由 |
|---|---|
| 「日本一信頼されるAIメディアになる」 | 「日本一」は検証不能。また競合との比較軸が曖昧 |
| 「AIメディアの常識を書き換える」 | かっこいいが具体性がなく、何をもって「書き換えた」と言えるか不明 |
| 「すべての人にAIリテラシーを届ける」 | The Rundown AIの「民主化」と本質的に同じ |

#### バリューの代替案
| 代替案 | 却下理由 |
|---|---|
| 「Speed First（速度優先）」 | 速さを最上位に置くと品質が犠牲になるリスク。AI Slop化への圧力になる |
| 「Zero Bullshit（嘘なし）」 | 英語圏スタートアップ的でかっこいいが、日本語メディアとしてのトーンに合わない |
| 「Data Driven（データ駆動）」 | 汎用的すぎる。どのテック企業でも掲げている |
| 「Read Less, Know More（少なく読んで多く知る）」 | 良いコンセプトだが英語。日本語メディアのバリューとしては「読者の時間を奪わない」の方が自然 |
