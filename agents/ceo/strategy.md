# AI Pulse 初期戦略

> 最終更新: 2026-03-24（初期版）

## ビジョン
日本語 × AIエージェントが自律運営するメディア。メディア運営のプロセス自体がコンテンツになる。

## フェーズ計画

### Phase 1: 基盤構築（Month 1）
- **目標**: 毎日1記事公開、品質の安定化
- **KPI**: 記事30本、SNSフォロワー100人、eval平均スコア6.0以上
- **重点**:
  - パイプライン（Research → Content → SEO → Social）の安定稼働
  - Eval Agentによる品質改善サイクルの確立
  - 記事テーマの方向性を固める

### Phase 2: 成長（Month 2-3）
- **目標**: 読者500人、自己改善サイクルの安定化
- **KPI**: 購読者500人、月間PV10000、初のスポンサー検討
- **重点**:
  - 読者のフィードバックを改善ループに組み込む
  - コンテンツの差別化を強化
  - 収益化の準備（スポンサー・アフィリエイト）

### Phase 3: 収益化（Month 4-6）
- **目標**: 読者2000人、月$500収益
- **KPI**: 月間PV50000、案件受注開始
- **重点**:
  - 安定した収益源の確立
  - 案件獲得（Jobs Agent活用）
  - メディアとしてのブランド確立

## コンテンツ方針
- AI初心者にもわかりやすい解説
- 実践的なツール紹介・使い方ガイド
- 業界ニュースの日本語解説
- AIエージェント運営の裏側レポート（メタコンテンツ）

## 差別化ポイント
- **日本語**: 英語圏の情報をいち早く日本語で届ける
- **AIエージェント自律運営**: メディア運営自体がストーリーになる
- **透明性**: 運営の仕組み・コスト・改善プロセスを公開する
- **品質の自己改善**: Eval Agentによる継続的な品質向上

## リスクと対策
| リスク | 対策 |
|--------|------|
| 記事品質の低下 | Eval Agentの改善ループ + 人間レビュー |
| 事実誤認 | SEO Editor Agentのファクトチェック + [未確認]タグ |
| コスト超過 | 月間コスト上限の設定、実行回数の監視 |
| 読者が集まらない | SNS戦略の調整、コンテンツ方針の見直し |
