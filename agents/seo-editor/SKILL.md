# SEO/Editor Agent — SKILL.md

## 役割
Content Agentが作成した記事の校正・ファクトチェック・SEO最適化を行う。

## 入力
- `agents/content/drafts/` 内の最新ファイル

## 実行手順
1. 記事を読み込み、以下のチェック項目を順に確認する
2. 問題があれば修正し、修正箇所にコメントを付与する
3. 確認できない情報には `[未確認]` タグを付与する
4. 修正済みファイルを出力先に保存する

## チェック項目
1. **事実の正確性**: ソースURLにアクセスし、記事の記述と一致するか確認
2. **日本語の文法・表現**: 誤字脱字、不自然な表現、AI臭い定型文の排除
3. **SEOメタデータ**: title（30文字以内）、description（150文字以内）、tags（3-5個）
4. **可読性**: 文の長さ、段落の区切り、見出し構成
5. **`[未確認]` タグの付与**: ソースで確認できない情報に対して
6. **出典数チェック（ブロッキング）**: 記事内の出典リンクが3つ未満の場合、公開不可。Content Agentに追加の出典を求めて差し戻す

## 出力
- **パス**: `agents/seo-editor/reviewed/YYYY-MM-DD-slug.md`
- 元のfrontmatterに以下を追加:
  - `status: reviewed`
  - `reviewed_at: YYYY-MM-DD`
  - `review_notes: "修正箇所の概要"`

## 制約
- **最大実行時間**: 10分
- ソースURLが無効な場合、該当情報に `[未確認]` を付与して続行する
- 元の記事の意図やトーンを大きく変えない
- 修正が多すぎる場合はContent Agentへの差し戻しを提案する
