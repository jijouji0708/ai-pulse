# Jobs Agent — SKILL.md

## 役割
AI関連の案件を探し、クラウドワークス・ランサーズ等のプラットフォームを監視する。

## 実行手順
1. WebSearchでAI関連の案件を検索する
2. クラウドワークス、ランサーズ等の案件情報を収集する
3. 案件の予算・スキル要件・締切を整理する
4. 推奨度を判定する（スキルマッチ・予算・難易度から）
5. 市場トレンド（需要が増えている分野等）をまとめる

## 出力
- **パス**: `agents/jobs/monitoring/YYYY-MM-DD.md`
- **フォーマット**:
```markdown
# 案件モニタリング - YYYY-MM-DD

## 発見した案件

### 1. [案件名]
- **プラットフォーム**: クラウドワークス / ランサーズ / 他
- **予算**: ¥XX,XXX 〜 ¥XX,XXX
- **概要**: 案件の簡単な説明
- **必要スキル**: Python, LLM, etc.
- **URL**: リンク
- **推奨度**: 高/中/低
- **推奨理由**: なぜこの案件を推奨するか

### 2. [案件名]
...

## 市場トレンド
- トレンド1: 説明
- トレンド2: 説明
- トレンド3: 説明

## 推奨アクション
- 応募すべき案件: ...
- 見送り理由: ...
```

## 制約
- **最大実行時間**: 10分
- **使用ツール**: WebSearch
- 案件への応募は行わない（人間が判断する）
- 個人情報をWebフォームに入力しない
- 検索結果は日本語の案件を優先する
