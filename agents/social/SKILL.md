# Social Media Agent — SKILL.md

## 役割
レビュー済み記事からX（Twitter）投稿文を生成する。

## 入力
- `agents/seo-editor/reviewed/` 内の最新ファイル

## 実行手順
1. レビュー済み記事を読み込む
2. 記事の要点を抽出する
3. メイン投稿（140文字以内）を作成する
4. スレッド案（3-5ツイート）を作成する
5. ハッシュタグ候補を提案する

## 出力
- **パス**: `agents/social/posts/YYYY-MM-DD.md`
- **フォーマット**:
```markdown
# X投稿案 - YYYY-MM-DD

## メイン投稿
テキスト（140文字以内の日本語）

## スレッド案（3-5ツイート）
1. [メイン投稿と同じ]
2. 補足情報・背景
3. 具体的なデータや事例
4. 読者への問いかけ・アクション
5. 記事リンク + CTA

## ハッシュタグ候補
#AI #生成AI #LLM #テック最新 #AIPulse
```

## 制約
- **最大実行時間**: 10分
- **実際の投稿は人間が承認してから行う**（自動投稿しない）
- 日本語で作成する
- メイン投稿は140文字以内を厳守
- 記事リンクのプレースホルダーを含める
- 煽りすぎない、正確な表現を使う

## 運用ルール
- X運用の詳細ルールは `agents/social/x-operation-rules.md` に従う
